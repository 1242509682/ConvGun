# ConvGun 微光转换枪

- 作者: 羽学、星梦
- 出处: TShock官方Q群816771079
- 这是一个 TShock 6.1.0 服务器插件，适用于 Terraria 1.4.5。
  主要功能：微光枪命中掉落物时，按规则转换，支持条件与动画；无规则时回退原版微光或分解

## 指令

| 语法                     | 权限               | 说明                               |
| ------------------------ | ------------------ | ---------------------------------- |
| /sg ls [页数]            | sg.use             | 列出转换规则（分页）                |
| /sg ls all               | sg.use             | 列出所有转换规则（全量）            |
| /sg if                   | sg.use             | 查看插件状态（开关、动画、规则数量）|
| /sg all                  | sg.use             | 查看所有可用条件                    |
| /sg a                    | sg.admin           | 开始交互式添加转换规则              |
| /sg d                    | sg.admin           | 删手持物品所有规则                  |
| /sg s                    | sg.admin           | 设置源物品（手持）                  |
| /sg sk <数量>            | sg.admin           | 设置转换数量（默认1）               |
| /sg cd                   | sg.admin           | 设置触发条件（如 晚上 血月 肉后）   |
| /sg lk                   | sg.admin           | 设置幸运值                          |
| /sg ok                   | sg.admin           | 完成并保存当前规则                  |
| /sg no                   | sg.admin           | 取消当前规则添加                    |
| /sg rs                   | sg.admin           | 清空所有规则                        |
| /sg fix                  | sg.admin           | 恢复默认规则（内置微光转换表）      |
| /reload                  | tshock.cfg.reload  | 重载配置文件                        |

> 基础权限 sg.use 管理权限 sg.admin

## 使用方法
1. 将插件放入 ServerPlugins 文件夹，重启服务器。
2. 给玩家权限:/group addperm default sg.use
3. 击败史莱姆王获取水枪,丢入微光湖 转换为 微光枪
4. 使用微光枪射击掉落物即可触发转换与分解功能

- 优化了cd指令,无参数支持显示所有条件
- 加入了d指令,自动移除所有手上物品相关的自定义规则
- 移除了t指令(冗余设计)
- 优化代码查找性能

v1.0.1
- 当没有自定义规则时自动使用原版微光转换与分解
- 加入了随机动画

v1.0.0
- 实现微光枪弹幕射击掉落物转换物品、怪物并实现粒子动画
- 支持物品/怪物转换规则
- 支持条件进度多达87项
- 支持飞行动画模式与无动画模式
- 支持堆叠数量处理
